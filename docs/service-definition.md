## 4. ターゲット顧客

### 業種・施設種別
- 特別養護老人ホーム（特養）
- 介護老人保健施設（老健）
- グループホーム
- デイサービス・デイケア
- 訪問介護事業所
- 小規模多機能型居宅介護
- 有料老人ホーム

### 課題・状況
- 求人媒体に出稿しているが応募数が少ない
- 自社サイトに採用ページがない、または機能していない
- 採用担当者がWeb制作の知識を持っていない
- 採用コストを抑えたい（媒体掲載費の削減）

### 予算・意思決定
- 単発50,000円の決済権限がある（施設長・管理者・法人担当者）
- 即断できる小〜中規模施設が主なターゲット

## 5. 私たちが売っているもの

**応募導線の整備** — 求職者が「見つけて → 読んで → 応募する」という流れを完結させる仕組み。

採用結果ではなく、**採用が起きやすい環境そのもの**を提供します。

---

## 6. 私たちが約束すること（できること）

- 応募フォームが正常に動作し、応募が受け取れる状態を作る
- スマートフォンで快適に閲覧できるページを作る
- JobPosting 構造化データにより Google しごと検索への表示可能性を高める
- 30日間、改善のためのサポートを提供する
- 納品チェックリストに基づき、品質を可視化して引き渡す
- 誇張のない、正確な求人情報を掲載する

---

## 7. 私たちが約束しないこと（できないこと）

- 採用成功・内定・採用人数の保証
- 応募者数の最低保証
- 求人媒体への掲載・掲載費の負担
- SNS・広告運用
- 写真・動画制作
- 既存人事システムとの連携
- 求人ページを作れば必ず採用できるという断言

---

## 8. 禁止表現（社内・対外問わず使用禁止）

以下の表現は、誇大広告・景品表示法違反・信頼損失につながるため、一切使用しない。

| 禁止表現 | 理由 |
|---------|------|
| 「採用できます」「採用成功をお約束します」 | 採用は保証不可 |
| 「必ず応募が増えます」 | 数値保証は不可 |
| 「○倍の応募が見込めます」 | 根拠のない数値保証 |
| 「最短1週間で採用」 | 具体的な成果保証 |
| 「他社より優れています」 | 根拠なき比較 |
| 「今だけ特別価格」（常設の場合） | 不当表示の可能性 |
| 「プロが作るから安心」 | 曖昧な品質保証 |
| 「AIが全自動で採用ページを作る」 | 誤解を招く表現 |

## 10. このサービスの差別化ポイント

1. **AI（Codex）を制作プロセスに組み込み、品質を可視化**
   - モック生成・コードレビューをAIで実施し、レポートとして納品

2. **介護業界に特化したコピーライティング**
   - 一般的なLP制作ではなく、介護求職者の行動・心理に合わせた文章

3. **構造化データ（JobPosting）の標準実装**
   - Googleしごと検索対応を標準で含む（多くの格安制作会社は省略）

4. **誠実な提案スタイル**
   - 採用保証をしない代わりに、できることを明確に伝える

5. **30日間のサポートつき**
   - 納品後のフォローがあることで、クライアントの安心感が高い
