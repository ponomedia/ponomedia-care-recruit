# 無料診断テンプレート — 採用ページ現状スコアリング

最終更新: 2026-05-17

---

## 診断概要

介護施設・事業所の採用ページ（自社サイト・採用LP・求人情報ページ）を、
応募導線の観点から14項目でスコアリングし、現状課題と改善提案をレポートとして提供します。

**満点：100点**
**診断所要時間：30〜60分**
**レポート送付：診断後3〜5営業日**

## 50,000円パックでの対応方針

本診断の結果をもとに、「介護採用まるごと導線パック（50,000円）」では以下の点を整備します：

| 診断で判明した課題 | パックでの対応 |
|------------------|--------------|
| （例）採用LPが存在しない | スマホ対応の採用LP（HTML）を新規制作 |
| （例）応募フォームがない | Google フォーム設置・自動返信設定 |
| （例）構造化データが未実装 | JobPosting JSON-LD の実装 |
| （例）OGPが設定されていない | OGPメタタグの実装 |
| （例）GA4が未設置 | GA4タグの設置・動作確認 |
