# care-recruit-growth — 介護採用まるごと導線パック 運用マスターガイド

---

## サービス概要

**介護採用まるごと導線パック**は、介護施設・事業所が抱える「求人を出しても応募が来ない」という課題に対し、
応募導線（ランディングページ・求人情報ページ・応募フォーム・構造化データ）を一式整備することで、
求職者が「見つけて → 読んで → 応募する」という流れをスムーズに完結させるサービスです。

採用成功そのものを保証するのではなく、**応募が発生しやすい環境を構築すること**に特化しています。

## ターゲット

| 項目 | 内容 |
|------|------|
| 業種 | 介護施設・訪問介護事業所・グループホーム・デイサービスなど |

| Web習熟度 | 低くても可（対応はこちらで完結） |
| 予算感 | 単発50,000円を許容できる事業所 |

## やること／やらないこと

### やること
- 採用ページの設計・コピー・実装・公開補助
- 求職者が応募しやすい導線の構築
- SEO基礎（構造化データ・OGP・タイトル最適化）
- 30日間の改善サポート

### やらないこと
- 採用成功・内定・採用人数の保証
- 求人媒体（Indeed・求人ボックスなど）への掲載代行
- 写真撮影・動画制作
- SNS運用・広告運用
- 求人媒体費用の負担
- 既存システムへの深い連携開発

## 制作フロー

```
問い合わせ受付
    ↓
無料診断（現状スコアリング）
    ↓
提案・見積提示
    ↓
契約・入金確認
    ↓
ヒアリング（オンライン30分）
    ↓
コピーライティング作成
    ↓
Codex imagegen によるモック生成
    ↓
モック確認・修正（クライアント承認）
    ↓
HTML/CSS 実装
    ↓
Codex コードレビュー
    ↓
修正・最終調整
    ↓
公開・納品
    ↓
30日間改善サポート
    ↓
事例化（任意）
```

詳細は `docs/workflow.md` を参照。
