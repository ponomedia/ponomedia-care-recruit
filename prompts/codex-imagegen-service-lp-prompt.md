# Codex 画像生成プロンプト
## サービス LP「介護採用まるごと導線パック」モック生成指示

---

## 生成目的

PonoMedia のサービス LP「介護採用まるごと導線パック」の **スマートフォン表示モック画像** を生成する。
生成された画像は Claude Code への HTML 実装指示に使用するため、
**実際に HTML 化できるレベルの具体的なレイアウト・コピー・UI 要素** を含めること。

---

## デザイン仕様

### カラーパレット
| 用途 | 色 |
|------|----|
| 背景（メイン） | 白（#FFFFFF） |
| 背景（サブ） | 淡いブルー（#EBF4FA） |
| 背景（アクセント背景） | 淡いグリーン（#EBF5F0） |
| アクセント・CTAボタン | オレンジ（#F47B20） |
| テキスト（メイン） | ダークグレー（#333333） |
| テキスト（サブ） | ミディアムグレー（#666666） |
| ボーダー | ライトグレー（#E0E0E0） |

### タイポグラフィ
- フォント：Noto Sans JP
- 見出し（h1）：24px / Bold
- 見出し（h2）：20px / Bold
- 本文：14px / Regular
- 小文字：12px / Regular

### デバイス
- スマートフォン（幅 375px）を基準にモックを生成
- 縦長スクロールページとして全セクションを縦に並べて表示

## 生成指示

- 各セクションを **縦に並べた 1 枚のスクロールスクリーンショット風画像** として生成する
- 実際のスマートフォン画面のように **幅375px、背景白のフレーム内** にデザインする
- フォントは日本語が明瞭に読めるクリーンなサンセリフ体
- アイコンは線画（ライン）スタイルで統一
- 余白（パディング）は各セクション上下 40px 程度
- **「採用保証」などの断定表現は一切入れない**
