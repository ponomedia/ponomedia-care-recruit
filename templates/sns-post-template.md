# SNS 投稿文テンプレート
## 採用目的 Instagram / X 両対応 全10本

---

## 使い方

1. `【】` で囲まれたプレースホルダーを施設情報に差し替える
2. Instagram：本文300字以内 + ハッシュタグ（本文の後に追加）
3. X（旧Twitter）：全体で140字以内になるようトリミングする
4. ハッシュタグは施設のアカウントに合わせて取捨選択する
5. 投稿前に「採用保証」「必ず採用」などの断定表現がないか確認する

## Post 10 — 応募促進（背中を押す）

### Instagram 用

```
「応募しようか迷っている」方へ、最後に一言。

迷っているうちは、見学に来てみてください。

履歴書なし・服装自由・気軽に来てOKです。
見学後に「やっぱり違うな」と思っても、
それはそれで良い情報だと思います。

私たちも、無理に採用したいわけではありません。
お互いが「ここで働いてみたい」「一緒に働いてほしい」
と思えることが大事だと考えています。

まず話を聞くだけでも、全然OKです。
DMまたはプロフィールのフォームからどうぞ📩

#迷ってるなら見学へ #介護転職 #介護求人 #見学受付中 #【施設名】 #【地域名】 #介護の仕事 #一緒に働こう
```

### X 用

```
迷っているなら、まず見学へ。履歴書なし・服装自由・気軽にOK。見学後に「違う」と思っても全然構いません。お互いが「ここがいい」と思えることが大事。DM or プロフリンクから☺️ #介護求人 #見学受付中 #介護転職
```
