# 面接日程調整テンプレート
## 5種：初回返信 / 日程確定 / 当日案内 / 面接後フォロー / 不採用通知

---

## 使い方

- `【】` のプレースホルダーを実際の情報に差し替えて送信する
- 不採用通知は特に文体に注意し、相手の尊厳を傷つけない表現を使う
- 送信前に宛先・氏名・日時・担当者名を必ず確認する

---

つきましては、面接の日程についてご調整させてください。

ご都合のよいものをお知らせください。

【候補日1】：〇月〇日（〇）〇時〜 または 〇時〜

【候補日3】：〇月〇日（〇）〇時〜 または 〇時〜

## テンプレート 3 — 当日案内（面接当日の朝に送信）

**用途**：面接当日の朝（8〜9時頃）に送信するリマインド

### LINE / ショートメッセージ版（短文推奨）

```
おはようございます。
【施設名】採用担当の【担当者名】です。

本日 【〇時〜】の面接の件、お待ちしております。

📍【施設名】（【住所・簡略版】）

ご不明な点や遅刻の際はお気軽にご連絡ください。
TEL：【電話番号】

それでは本日、どうぞよろしくお願いいたします。
```

考えておりますので、内定のご連絡をいたします。

下記の日程でご都合いかがでしょうか。

ご検討のほどよろしくお願いいたします。

━━━━━━━━━━━━━━━━

## テンプレート 5 — 不採用通知（丁寧な文体）

**注意事項**
- 不採用理由の詳細は原則記載しない（トラブル防止）
- 「今回は縁がなかった」「ご縁があれば」などの表現は使わない
- 応募者の人格・能力を否定する表現は絶対に使わない
- 可能であれば、別ポジションやタイミングへの案内を含める

### メール版

```
件名：選考結果のご連絡｜【施設名】採用担当

【氏名】様

先日はお忙しい中、面接にお越しいただきありがとうございました。
【施設名】採用担当の【担当者名】です。

この度は、慎重に選考を行いました結果、
誠に恐れ入りますが、今回は採用を見送らせていただくことになりました。

【氏名】様のこれまでのご経験・ご経歴は大変魅力的に感じておりましたが、
今回の募集条件との兼ね合いにより、このような結果となりました。

ご応募いただいたことに心より感謝申し上げます。
今後のご活動において、良いご縁がございますよう
心よりお祈り申し上げます。

なお、本件についての詳細なご説明は
ご対応が難しい状況にありますこと、
どうかご了承いただけますと幸いです。

ありがとうございました。

━━━━━━━━━━━━━━━━
【施設名】採用担当
【担当者名】
TEL：【電話番号】
MAIL：【メールアドレス】
━━━━━━━━━━━━━━━━
```

### LINE / ショートメッセージ版

```
【氏名】様

先日はお越しいただきありがとうございました。
【施設名】採用担当の【担当者名】です。

選考の結果、誠に恐れながら、
今回は採用を見送りとなりました。

ご応募いただいたことを感謝しております。
今後のご活動がうまくいくことを願っております。

ありがとうございました。
```
