# 求人文テンプレート
## ハローワーク版 / Indeed版 / Airワーク版

---

## 使い方

1. 掲載する媒体のテンプレートを選択する
2. `【】` で囲まれたプレースホルダーをクライアント情報に差し替える
3. 文字数カウントで各媒体の制限内に収まっているか確認する
4. 禁止表現リストで問題のある表現がないか最終チェックする

# A. ハローワーク版

## 文字数・制限

| 項目 | 制限 |
|------|------|
| 仕事内容 | 500字以内 |
| 事業内容 | 150字以内 |
| 会社の特長 | 350字以内 |
| 求人のポイント | 各50字以内（最大10項目） |

## 推奨表現
- 「未経験歓迎」「ブランクOK」「資格取得支援あり」
- 「アットホームな職場」「スタッフ定着率○%」
- 「土日祝休み」「残業少なめ」「シフト相談OK」

## 禁止表現（ハローワーク）
- 性別・年齢による制限表現（例：「若い方歓迎」は不可）
- 「急募」の多用（根拠なき緊急性演出は不可）
- 採用保証・収入保証の表現

---

## テンプレート本文

### 求人区分
```
【介護 / 医療・福祉関連職種】
```

### 職種名
```
介護職員（【正社員 / パート・アルバイト】）
```

### 仕事内容
```
【施設名】は、【所在地・地域名】にある【施設種別（例：通所介護施設）】です。
定員【定員数】名の利用者様に、日常生活のサポートをお届けしています。

【担当業務の内容】
・食事の準備・配膳・食事介助
・入浴介助（【機械浴 / 個浴 / 両方】対応）
・排泄介助・おむつ交換
・レクリエーションの企画・実施
・送迎業務（【あり / なし】）
・記録業務（【紙 / タブレット】入力）

【職場の特徴を1〜2文で】
例：「残業はほとんどなく、定時退勤を推奨しています。」
例：「スタッフ間の情報共有を大切にし、困ったときはすぐ相談できる環境です。」

未経験・ブランクのある方も、先輩スタッフが丁寧にサポートしますのでご安心ください。
```

### 事業内容
```
【運営法人名】が運営する【施設種別】です。
【地域名】の地域に密着し、【設立年 / 創業年】から【利用者や地域への貢献を1文で】。
```

### 会社の特長
```
【施設名】の3つの特長をご紹介します。

1.【特長タイトル1】
【説明 50〜70字】

2.【特長タイトル2】
【説明 50〜70字】

3.【特長タイトル3】
【説明 50〜70字】

【働きやすさや待遇の補足を1〜2文】
```

### 求人のポイント（各50字以内・最大10項目）
```
1. 未経験・ブランクOK！丁寧な研修でゼロから始められます
2. 残業少なめ・定時退勤推奨
3. 資格取得支援あり（【資格名】の費用補助）
4. シフト相談OK・【週〇日〇時間〜】など柔軟な働き方可
5. 社会保険完備・交通費全額支給
6. アットホームな職場・スタッフの定着率が高い
7. 【施設のアピールポイント（例：夜勤なし / 土日休み）】
```

# B. Indeed 版

## 文字数・制限

| 項目 | 制限 |
|------|------|
| 求人タイトル | 100字以内（推奨50字以内） |
| 職種 | 選択式（カテゴリ選択） |
| 仕事内容 | 5,000字以内（推奨2,000字以内） |
| 会社の説明 | 1,000字以内 |

## Indeed特有の注意事項
- 求人タイトルに給与・ボーナス・時給等の数値を入れない（ガイドライン違反）
- キーワードの詰め込みは重複チェックで弾かれる場合あり
- 「#1」「最高の職場」等の根拠のない最上級表現は禁止

## 推奨キーワード（本文中に自然な形で含める）
`介護職員` `デイサービス` `訪問介護` `未経験歓迎` `ブランクOK` `介護福祉士` `初任者研修` `パート` `正社員` `シフト制` `日勤のみ`

---

## テンプレート本文

### 求人タイトル
```
【施設名】の介護職員（【正社員 / パート】）募集｜【地域名】・【特徴1語（例：日勤のみ）】
```

### 仕事内容（詳細版）
```
◆ 仕事の概要

【施設名】は【地域名】にある【施設種別】です。
私たちと一緒に、利用者様の「安心」と「笑顔」を届けませんか？

◆ 具体的な仕事内容

・食事介助（準備・配膳・食事サポート・口腔ケア）
・入浴介助（【機械浴 / 個浴 / 両対応】）
・排泄介助・おむつ交換
・移乗・移動の介助
・レクリエーションの実施（【例：体操・ゲーム・季節イベント等】）
・送迎業務（【あり・なし・担当者のみ】）
・記録業務（【タブレット / 紙】）

◆ こんな方に来てほしい

・介護のお仕事が初めての方
・ブランクがあって復帰を考えている方
・「人の役に立つ仕事」がしたい方
・働く時間を柔軟に調整したい方（※パート募集の場合）
・資格を取りながら長く働きたい方

◆ 先輩スタッフからひとこと

「【スタッフコメント（架空でも可能・クライアント確認後）】」

◆ 職場の雰囲気

【施設の雰囲気を2〜3文で。具体的なエピソードがあると良い。】

◆ 未経験でも安心の理由

・入職後【〇週間】は先輩スタッフがマンツーマンでサポート
・定期的な1on1面談で悩みを相談できる
・【資格名】の取得を会社が支援（費用補助あり）

◆ よくある質問

Q: 夜勤はありますか？
A: 【あり（月〇回程度） / なし（日勤のみ）】

Q: 週何日から働けますか？
A: 週【〇日〜】から勤務可能です。

Q: 資格がなくても応募できますか？
A: はい、資格不問で応募いただけます。入職後に資格取得を目指すことも可能です。
```

### 会社の説明
```
【運営法人名】は、【設立年 / 創業年】から【地域名】で介護サービスを提供してきました。

【施設の理念・大切にしていることを2〜3文】

現在、私たちのチームには【職員数】名のスタッフが在籍しています。
【職場の特徴や強みを1〜2文】
```

# C. Airワーク（リクルート）版

## 文字数・制限

| 項目 | 制限 |
|------|------|
| キャッチコピー | 40字以内 |
| 仕事内容 | 10,000字以内 |
| 応募要件 | 2,000字以内 |
| 会社・職場の魅力 | 10,000字以内 |

## Airワーク特有の注意事項
- 画像・動画を積極的に活用（採用ページとしての見栄えを重視）
- 求人タイトルはシンプルに職種名を基本とする
- 「採用お祝い金」等の特典表記はリクルート規約を確認の上使用

---

## テンプレート本文

### キャッチコピー（40字以内）
```
パターンA：「未経験OK！【地域名】で"ありがとう"を仕事に」
パターンB：「一緒に働きませんか？アットホームな【施設種別】」
パターンC：「【特徴（例：夜勤なし・日勤のみ）】の介護職員を募集中」
```

### 仕事内容
```
【施設名】では、一緒に働く仲間を募集しています。

▼ お任せする仕事
【業務内容を箇条書きで5〜8項目】

▼ 1日の流れ（例）
【時刻】　【業務内容】
【時刻】　【業務内容】
【時刻】　【業務内容】
（以下続く）

▼ 正直な話：大変なこともあります
【大変な点を1〜2項目、誠実に記載】
それでも、【やりがいを感じる瞬間を1文で】。

▼ 研修・サポート体制
【研修内容・OJT期間・定期面談などを記載】
```

### 会社・職場の魅力
```
▼ 私たちの職場の特徴

【特徴1のタイトル】
【説明100〜150字】

【特徴2のタイトル】
【説明100〜150字】

【特徴3のタイトル】
【説明100〜150字】

▼ スタッフの声
「【コメント（クライアント確認後に実名または匿名で）】」
（【年代・職種】）

▼ 働く環境
【社内の写真説明・設備・休憩スペース等を記載】
```

## 共通 禁止表現チェックリスト

出稿前に以下を必ず確認してください。

- [ ] 「採用保証」「採用率○%」などの断定的成果表現がない
- [ ] 「業界最安値」「No.1」などの根拠のない最上級表現がない
- [ ] 性別・年齢による制限・優遇表現がない（例：「若い方歓迎」）
- [ ] 「未経験可」と書いた場合、実際に未経験者を採用している
- [ ] 給与・勤務条件が実際と異なる記載になっていない
- [ ] 「待遇：交渉による」は原則使用しない（具体的金額を記載）
- [ ] プライバシーポリシーへのリンクが存在する（LP・応募フォームに）
