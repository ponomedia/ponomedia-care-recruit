# 応募後返信テンプレート
## 3種：応募お礼 / 見学案内 / 面接前日リマインド

---

## 使い方

- `【】` のプレースホルダーを実際の情報に差し替えて送信する
- メール・LINE・ショートメッセージいずれにも使用可能
- 送信前に宛先・日時・担当者名を必ず確認する
- 「採用保証」など断定的表現がないことを確認する

---

---

## 返信テンプレート 1 — 応募お礼

**用途**：応募受信後、できるだけ当日中（最長でも翌営業日）に送信

### メール版

```
件名：ご応募ありがとうございます｜【施設名】採用担当

【氏名】様

この度は【施設名】の求人にご応募いただき、
誠にありがとうございます。

採用担当の【担当者名】と申します。

ご応募内容を確認させていただきました。
改めて書類を拝見した上で、
【〇営業日以内（例：3営業日以内）】に
次のご連絡をさせていただきます。

少々お待ちいただけますと幸いです。

なお、ご質問やご不明な点がございましたら
お気軽にご返信ください。

引き続きよろしくお願いいたします。

━━━━━━━━━━━━━━━━
【施設名】採用担当
【担当者名】
TEL：【電話番号】（受付時間：平日 【〇時〜〇時】）
MAIL：【メールアドレス】
━━━━━━━━━━━━━━━━
```

### LINE / ショートメッセージ版

```
【施設名】採用担当の【担当者名】です。

この度はご応募いただき、ありがとうございます！
内容を確認いたしまして、【〇営業日以内】に改めてご連絡いたします。

ご質問がある場合はいつでもメッセージをお送りください。
どうぞよろしくお願いいたします。
```

---

### 返信が遅れた場合のお詫び文（追加）

```
（メール冒頭に追加）
ご返信が遅くなりまして、大変失礼いたしました。
【理由（例：連休のため / 担当者不在のため）】により、
ご連絡が遅れてしまいました。
改めてお詫び申し上げます。
```

## 返信テンプレート 2 — 見学案内

**用途**：見学日程が確定した際に送信

### メール版

```
件名：見学日程のご確認｜【施設名】採用担当

【氏名】様

お世話になっております。
【施設名】採用担当の【担当者名】です。

先日はご連絡いただきありがとうございました。
見学の日程が確定いたしましたので、ご案内いたします。

━━━━━━━ 見学日程 ━━━━━━━
日時：【〇月〇日（〇）〇時〜〇時（約60〜90分）】
場所：【施設名】
　　　〒【郵便番号】【住所】
　　　【最寄り駅からのアクセス】
担当：【担当者名】
━━━━━━━━━━━━━━━━━━

持ち物：特にございません。
服装：自由（動きやすい格好でお越しください）
履歴書：不要です（お持ちいただく必要はありません）

駐車場：【あり（無料）/ なし（近隣のコインパーキングをご利用ください）】

当日は【担当者名】がご案内いたします。
何かご不明な点がございましたら、お気軽にご連絡ください。

お会いできることを楽しみにしております。

━━━━━━━━━━━━━━━━
【施設名】採用担当
【担当者名】
TEL：【電話番号】（受付時間：平日 【〇時〜〇時】）
MAIL：【メールアドレス】
━━━━━━━━━━━━━━━━
```

### LINE / ショートメッセージ版

```
【施設名】採用担当の【担当者名】です。

見学の日程が確定いたしました。
ありがとうございます！

📅 日時：【〇月〇日（〇）〇時〜】
📍 場所：【施設名】【住所（簡略版）】

服装自由・履歴書不要・お気軽にどうぞ。
駐車場：【あり / なし】

当日ご不明な点があればいつでもご連絡ください。
お会いできることを楽しみにしています！
```

## 返信テンプレート 3 — 面接前日リマインド

**用途**：面接前日の夕方（17〜19時頃）に送信

### メール版

```
件名：明日の見学・面接のご確認｜【施設名】採用担当

【氏名】様

お世話になっております。
【施設名】採用担当の【担当者名】です。

明日の見学・面接についてリマインドのご連絡です。

━━━━━━━ 明日の予定 ━━━━━━━
日時：【〇月〇日（〇）〇時〜】
場所：【施設名】
　　　〒【郵便番号】【住所】
担当：【担当者名】
━━━━━━━━━━━━━━━━━━

当日は【担当者名】が受付でお迎えいたします。
到着された際は「採用の見学で参りました」と
受付スタッフにお声がけください。

もしご都合が変わった場合は、
お早めに以下の連絡先までご連絡ください。

TEL：【電話番号】
MAIL：【メールアドレス】

それでは明日、お待ちしております。

━━━━━━━━━━━━━━━━
【施設名】採用担当
【担当者名】
━━━━━━━━━━━━━━━━
```

### LINE / ショートメッセージ版

```
【施設名】採用担当の【担当者名】です。

明日の見学・面接についてリマインドです。

📅 明日 【〇時〜】
📍 【施設名】（【住所・簡略版】）

到着後は受付で「採用の見学で来ました」とお声がけください。
ご都合が変わった場合はご連絡いただけると助かります。

明日お会いできることを楽しみにしています！
```
