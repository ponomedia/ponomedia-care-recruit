# 採用 LP コピーテンプレート
## 介護採用まるごと導線パック / 施設別カスタマイズ用

---

## 使い方

1. `【】` で囲まれたプレースホルダーをクライアント情報に差し替える
2. 各セクションのコピーを実際の施設情報に合わせてカスタマイズする
3. 「大変なことも正直に」のセクションは必ず誠実な内容を記載する
4. 採用保証・断定的成果表現は使用しない

### h2 見出し

```

```

### 本文（誠実なトーンを維持すること）

```
【施設名】は、求職者の方に「入職後のイメージとのギャップ」をなくしてほしいと考えています。
だから、大変なこともきちんとお伝えします。

・【大変な点1（例：身体介助があるため、一定の体力が必要です）】
・【大変な点2（例：利用者様の体調変化に柔軟に対応することが求められます）】
・【大変な点3（例：チームでの連携が欠かせないため、コミュニケーションが重要です）】

それでも、【施設名】を選んで「よかった」と感じているスタッフが大多数います。
少しでも気になった方は、まず見学からお気軽にどうぞ。
```

## Section 10 — FAQ

### h2 見出し

```
よくあるご質問
```

### Q&A（6問）

```
Q1. 介護が未経験でも応募できますか？
A1. はい、ぜひご応募ください。現在のスタッフの約【〇割】が未経験でスタートしています。入職後の研修制度で丁寧にサポートします。

Q2. まず見学だけでもできますか？
A2. もちろんです。見学・面談は無料で行っています。「どんな雰囲気か見てから考えたい」という方も大歓迎です。

Q3. 夜勤はありますか？
A3. 【ありません（日勤のみ）。 / 月〇回程度あります。最初は夜勤なしで慣れることも可能です。】

Q4. 子育て中でも働けますか？
A4. はい。現在も【〇名】の子育て中スタッフが在籍しており、シフト調整や急な休みにも互いにフォローし合う文化があります。

Q5. 資格がなくても採用してもらえますか？
A5. 資格がなくてもご応募いただけます。入職後に資格取得を目指す方向けの費用補助制度もあります。

Q6. 転職が初めてで応募の仕方がわかりません。
A6. このページの応募フォームからお気軽にご連絡ください。「どんな仕事か聞いてみたい」程度でも問題ありません。
```
